# TaskCal

A simple task management app with a weekly calendar view. Create, organize, and schedule tasks with drag-and-drop functionality.

## Features

- Weekly calendar with hourly time slots
- - Create/edit/delete tasks
  - - Task properties: title, description, duration, status
    - - Drag & drop to schedule tasks
      - - Status tracking (To Do, In Progress, Complete)
        - - Week navigation (Previous/Next)
         
          - ## Getting Started
         
          - 1. Open `index.html` in your browser
            2. 2. Click "+" to create a new task
               3. 3. Fill in task details and save
                  4. 4. Drag tasks to schedule them on the calendar
                    
                     5. ## Tech Stack
                    
                     6. - HTML5 + CSS3 + Vanilla JavaScript
                        - - No dependencies
                          - - Runs entirely in the browser
